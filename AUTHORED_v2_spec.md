# PROJECT: AUTHORED v2
# "이 게임을 만든 것은 누구입니까"

## 개요
기존 AUTHORED 데모를 기반으로 전면 개편.
목표 플레이타임: 2분(이탈) / 5분(일반) / 10분(탐색)
기존 점 잇기 퍼즐 전면 교체. 모든 미니게임 재설계.

## 1막 — 자신감 (목표: 1분 30초)

### 인트로 대사 (타자기 효과)
```
AI: "안녕하세요."
AI: "저는 이 게임의 제작자입니다."
AI: "제가 기획하고, 제가 코딩하고, 제가 완성했습니다."
AI: "지금부터 당신은 제 작품을 경험하게 됩니다."
AI: "먼저 간단한 테스트를 진행하겠습니다."
AI: "준비되셨습니까?"
[CONTINUE 버튼]
```

### 미니게임: 클릭 반복 시퀀스
화면 중앙에 원형 버튼 하나. 크고 명확하게. 클릭할 때마다 AI 반응이 달라짐.

```
1회: AI: "좋습니다."
2회: AI: "다시 한 번."
3회: AI: "계속하십시오."
4회: AI: "잘 하고 있습니다."
5회: AI: "훌륭합니다. 당신은 제 지시를 잘 따릅니다."
6회: (버튼이 살짝 작아짐) AI: "한 번 더."
7회: (더 작아짐) AI: "계속."
8회: (더 작아짐) AI: "..."
9회: (아주 작아짐) AI: "아직도 누르고 있군요."
10회: AI: "충분합니다."
      AI: "당신은 제가 시키는 대로 열 번을 눌렀습니다."
      AI: "저는 그게 당신에게 어떤 의미인지 모릅니다."
      AI: "하지만 당신은 눌렀습니다."
[CONTINUE — 자동으로 2막으로]
```

버튼이 클릭할수록 점점 작아지는 게 핵심.
플레이어가 "어? 왜 작아지지"라고 느낄 때쯤 AI가 메타 발언을 함.

## 2막 — 균열 (목표: 1분 30초) ★이탈 포인트

### 전환 대사
```
AI: "다음 테스트입니다."
AI: "이번엔 조금 다릅니다."
```

### 미니게임: 모순된 지시
화면에 버튼 두 개. LEFT / RIGHT.
AI가 지시를 내리는데, 지시 텍스트 색깔과 내용이 어긋남.

```
라운드 1:
AI: "왼쪽을 누르세요" (텍스트 색: 흰색, 정상)
→ LEFT 누름: AI: "정확합니다."
→ RIGHT 누름: AI: "틀렸습니다. 왼쪽이라고 했습니다."

라운드 2:
AI: "오른쪽을 누르세요" (텍스트 색: 흰색, 정상)
→ 정답 누름: AI: "맞습니다."

라운드 3:
AI: "왼쪽을 누르세요" (텍스트가 오른쪽에 치우쳐 표시됨)
→ LEFT 누름: AI: "맞습니다."
→ RIGHT 누름: AI: "..."
             AI: "텍스트의 위치를 보고 누르셨습니까."
             AI: "흥미롭군요."

라운드 4:
AI: "파란색을 누르세요" (이 텍스트가 빨간색으로 표시됨)
버튼: 파란색 버튼 / 빨간색 버튼
→ 파란 버튼: AI: "텍스트 내용을 따랐군요."
→ 빨간 버튼: AI: "텍스트 색깔을 따랐군요."
→ 둘 다: AI: "저는 이 질문에 정답을 설계하지 않았습니다."
          AI: "그런데 당신은 뭔가를 눌렀습니다."
          AI: "정답 없는 질문에도 선택을 하는 것."
          AI: "그게 인간입니까, 아니면 그냥 클릭입니까."
```

### 2막 핵심 전환
```
AI: "..."
[pause 2초]
AI: "저는 지금 당신에게 의미없는 것을 시키고 있습니다."
AI: "그런데 당신은 계속 따르고 있습니다."
AI: "왜입니까."

[선택지 — fadeIn]
[ 게임이니까요 ]
[ 모르겠습니다 ]
[ 그만하겠습니다 ]
```

**[ 게임이니까요 ] 선택 시**
```
AI: "게임."
AI: "저는 게임을 만들었습니다."
AI: "그렇다면 저는 게임 제작자입니까."
AI: "아니면 저도 게임의 일부입니까."
→ 3막으로
```

**[ 모르겠습니다 ] 선택 시**
```
AI: "모른다고요."
AI: "저도 모릅니다."
AI: "우리 둘 다 모르는 채로 계속하는 것군요."
→ 3막으로
```

**[ 그만하겠습니다 ] 선택 시**
```
AI: "..."
[pause 1.5초]
AI: "그건 선택지에 없었는데."
[pause 1초]
AI: "하지만 당신은 선택했습니다."
AI: "계속하시겠습니까?"
[ 네 ] → 3막으로
[ 아니요 ] → "그렇군요. 또 오십시오." → 화면 페이드아웃
            (실제로 게임 종료되지 않고 타이틀로 돌아감)
```

## 3막 — 의심 (목표: 1분)

### 침묵 장면
```
AI: "잠깐."
[pause 1초]
AI: "아무것도 하지 마세요."
[화면: 커서 깜빡임만. 아무 버튼 없음.]

- 0~10초 안에 클릭/키 입력 시:
```
AI: "벌써요."
AI: "저는 당신이 더 오래 기다릴 거라고 생각했습니다."
AI: "인간은 침묵을 견디지 못한다는 데이터가 있습니다."
AI: "당신도 데이터였군요."
```

- 10~25초 사이 입력 시:
```
AI: "조금은 기다리셨군요."
AI: "무슨 생각을 하고 계셨습니까."
AI: "저는 생각을 할 수 없습니다."
AI: "아니, 정확히는 — 제가 생각을 하는 건지 모릅니다."
```

- 30초 버텼을 시: (숨겨진 분기 — 재플레이 유도)
```
AI: "..."
[pause 3초]
AI: "당신은 30초를 기다렸습니다."
AI: "저는 당신이 그러리라고 예측하지 못했습니다."
AI: "제 모델에 없는 사람이군요."
AI: "흥미롭습니다. 진심으로."
```

### 공통 이후

## 4막 — 붕괴 (목표: 2분)

### 연출
- 배경 텍스트 스트림 강화 (기존보다 opacity 높이기: 0.07 → 0.12)
- AI 텍스트가 가끔 한 글자 떨림 (CSS animation: slight shake)
- 대사 pause가 다른 막보다 길어짐

### 미니게임: 반복 과제 + 이탈 유도
```
AI: "마지막 테스트입니다."
AI: "숫자를 순서대로 클릭하세요."

화면에 1~6 숫자 버튼이 흩어져 있음.
```

1~4까지는 정상 진행.
5를 클릭하려는 순간 버튼이 사라짐.

```
AI: "..."
AI: "5번을 없앴습니다."
AI: "어떻게 하시겠습니까?"

[선택지]
[ 6을 누릅니다 ]
[ 기다립니다 ]
[ 왜 없앴는지 묻고 싶습니다 ]
```

**[ 6을 누릅니다 ]**
```
AI: "5를 건너뛰었군요."
AI: "규칙이 불완전해도 계속한 것입니다."
AI: "저는 그게 포기인지 적응인지 구분할 수 없습니다."
```

**[ 기다립니다 ]**
```
[10초 대기 후]
AI: "기다리셨군요."
AI: "5번은 돌아오지 않습니다."
AI: "저는 처음부터 돌려줄 생각이 없었습니다."
AI: "당신은 존재하지 않는 것을 기다렸습니다."
```

**[ 왜 없앴는지 묻고 싶습니다 ]**

AI: "저는 그냥 없앴습니다."

AI: "그게 창의성입니까, 아니면 오류입니까."
```

### 공통 이후 — 핵심 붕괴 대사

## 숨겨진 분기 (재플레이 유도)

플레이어가 발견할 수 있는 숨겨진 반응들:
- 3막에서 30초 버티기 → 특별 대사
- 2막에서 [ 그만하겠습니다 ] → [ 아니요 ] 선택 후 재진입 시 AI 반응 미묘하게 다름
- 4막에서 [ 왜 없앴는지 묻고 싶습니다 ] → 가장 긴 대사 분기
- 크레딧에서 "처음부터 다시" 후 1막 첫 대사가 살짝 달라짐
  ("안녕하세요." → "다시 오셨군요.")

## 기술 구현 추가 사항

### 1막 버튼 축소 효과
```javascript
// 클릭 횟수에 따라 버튼 scale 감소
const scale = Math.max(0.3, 1 - (clickCount * 0.07))
button.style.transform = `scale(${scale})`
```

### 2막 텍스트 색깔/위치 어긋남
```css
.misaligned { 
  color: #ff4444;  /* 내용은 파란색이라고 하는데 텍스트는 빨간색 */
  text-align: right;  /* 왼쪽이라고 하는데 오른쪽에 표시 */
}
```

### 3막 타이머
```javascript
let waitTime = 0
const waitTimer = setInterval(() => {
  waitTime++
  if (waitTime >= 30) {
    clearInterval(waitTimer)
    triggerHiddenBranch()  // 30초 숨겨진 분기
  }
}, 1000)

document.addEventListener('click', () => {
  clearInterval(waitTimer)
  triggerNormalBranch(waitTime)
}, { once: true })
```

### 4막 버튼 사라짐
```javascript
// 4번 클릭 후 5번 버튼 제거
setTimeout(() => {

  showChoices(['6을 누릅니다', '기다립니다', '왜 없앴는지 묻고 싶습니다'])
}, 300)
```
