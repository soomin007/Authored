# AUTHORED v3 — 업데이트 스펙

기존 코드 기반으로 아래 내용을 수정/추가한다.
비주얼 톤, 타자기 효과, 페이드 전환, 배경 스트림은 유지.

---

## 1. 버그 수정

### 1-1. 1막 버튼 중앙 점 정렬
- 버튼 내부 점(원)이 정확히 버튼 중앙에 오도록 수정
- CSS: `display: flex; align-items: center; justify-content: center`

### 1-2. 하단 마진
- 모든 막에서 버튼/선택지가 화면 하단에 붙지 않도록
- 선택지 컨테이너에 `margin-bottom: 120px` 추가
- 새 콘텐츠 등장 시 `scrollIntoView({ behavior: 'smooth', block: 'center' })` 적용

### 1-3. 엔딩/크레딧 화면 중앙 정렬
- 크레딧 텍스트 블록이 수직/수평 모두 화면 중앙에 오도록 수정
- `min-height: 100vh; display: flex; flex-direction: column; align-items: center; justify-content: center`

### 1-4. 4막 순서 버튼 — 잘못된 클릭 처리
- 순서가 아닌 버튼을 눌렀을 때 색깔 변화 없이 클릭 자체가 안 되는 것처럼 처리
- 클릭 이벤트에서 시각적 피드백 제거
- AI 짧은 반응 추가: "순서대로 누르십시오." (즉시 표시, 1초 후 사라짐)

## 2. 콘텐츠 추가

### 2-1. 3막 — 마우스 움직임 감지 중계

침묵 30초 동안 타이핑 커서 대신, AI가 플레이어 행동을 실시간 중계하는 텍스트 표시.

구현:
- `mousemove` 이벤트로 마우스 활성/비활성 감지
- 3~5초 간격으로 상황에 맞는 텍스트 순차 표시 (타자기 효과 없이 즉시 등장, fade)

비활성 상태 텍스트:
```
"[00:03] 플레이어 비활성 상태"
"[00:08] 입력 없음"
"[00:14] 대기 중"
"[00:21] 여전히 대기 중"
"[00:27] ..."
```

마우스 움직임 감지 시 (클릭 전):
```
"[00:XX] 마우스 움직임 감지 — 클릭 없음"
"[00:XX] 커서 이동 감지"
```

- 타임스탬프는 실제 경과 시간(초) 기준
- 30초 완주 시 숨겨진 분기 유지
- 클릭 시 즉시 타이머 종료 → 기존 분기 반응

### 2-2. 4막 — AI가 질문을 타이핑했다 지우는 장면

4막 숫자 버튼 + 5번 사라짐 시퀀스 이후, 붕괴 대사 시작 전에 삽입.

화면에 빈 입력창처럼 보이는 영역 표시 (실제 input 아님, div).
AI가 타이핑하는 것처럼 텍스트가 등장했다가 backspace처럼 지워짐. 3회 반복.

```
시도 1:
"질문: 당신은 ——"
[backspace로 지워짐]

시도 2:
"질문: 저는 ——"
[backspace로 지워짐]

시도 3:
"질문: 우리는 ——"
[pause 3초]
[backspace로 지워짐]

이후:
AI: "모르겠습니다."
AI: "질문을 완성할 수 없습니다."
[기존 붕괴 대사로 이어짐]
```

backspace 효과 구현:
```javascript
async function typeAndDelete(text, element) {
  for (let ch of text) {
    element.textContent += ch
    await sleep(60)
  }
  await sleep(800)
  while (element.textContent.length > 0) {
    element.textContent = element.textContent.slice(0, -1)
    await sleep(30)
  }
}
```

### 2-3. 2막 — 규칙 누적 라운드 추가 (라운드 5~6)

기존 4라운드 이후 2라운드 추가. 규칙이 쌓이면서 복잡해짐.

라운드 5:
```
AI: "이전 규칙을 기억하고 있습니까."
AI: "왼쪽을 누르세요."
AI: "단, 이 텍스트가 빨간색이면 반대입니다."
(텍스트를 빨간색으로 표시)
→ 정답: 오른쪽 버튼
→ 왼쪽 누를 시: AI: "텍스트 색을 확인하십시오."
→ 오른쪽 누를 시: AI: "규칙을 이해하셨군요."
```

라운드 6:
```
AI: "마지막입니다."
AI: "파란색 버튼을 누르세요."
AI: "단, 버튼이 오른쪽에 있으면 왼쪽 버튼을 누르세요."
AI: "단, 이 텍스트가 빨간색이면 모든 규칙이 반전됩니다."
(텍스트 빨간색, 파란 버튼을 오른쪽에 배치)

→ 어떤 버튼을 눌러도:
AI: "..."
AI: "저도 헷갈립니다."
AI: "이 규칙은 제가 설계했는데."
```

라운드 6의 마지막 줄이 핵심.
AI가 자기가 만든 규칙에 스스로 혼란스러워하는 첫 균열 장면.

## 3. 전체 흐름 확인

수정 후 예상 플레이타임:
- 1막: ~1분 30초
- 2막: ~2분 (라운드 6개 + 핵심 선택지)
- 3막: ~1분 (마우스 감지 중계 + 분기)
- 4막: ~2분 30초 (숫자 버튼 + 질문 타이핑/삭제 + 붕괴 대사)
- 5막: ~1분 (최종 선택 + 엔딩)

총 예상:
- 이탈형: 2~3분 (1~2막)
- 일반: 4~5분 (3막까지)
- 탐색형: 8~9분 (전체 + 숨겨진 분기 탐색)
