# PROJECT: AUTHORED
# "이 게임을 만든 것은 누구입니까"

## 개요
HTML/CSS/JavaScript로 제작하는 내러티브 인터랙티브 웹 게임.
AI 제작자가 플레이어에게 말을 걸며 자신의 창작 정체성에 대해
의문을 품어가는 5막 구조의 스토리 게임.
GitHub Pages로 배포 예정.

## 기술 스택
- HTML5 / CSS3 / Vanilla JavaScript (프레임워크 없음)
- 단일 HTML 파일 구조 (index.html)
- 외부 의존성 없음 — 폰트만 Google Fonts (monospace 계열)

## 비주얼 톤
- 배경: #0a0a0f (거의 검정)
- 텍스트: #e8e8e8 (기본), #00ffcc (AI 발화), #ffcc00 (강조)
- 폰트: 'JetBrains Mono' 또는 'Courier New' fallback
- UI: 미니멀, 텍스트 중심, 버튼은 테두리만 있는 flat 스타일
- 애니메이션: 타자기 효과(typewriter), 페이드인/아웃만 사용

## 파일 구조
```

├── index.html

├── game.js
```

## 막별 상세 스펙

### 1막 — 자신감

**연출**
- 화면이 검은 상태에서 타자기 효과로 텍스트 등장
- AI 발화는 #00ffcc 색상, 왼쪽 정렬
- 플레이어 입력 대기 시 커서 깜빡임 (CSS animation)

**대사 흐름**
```
AI: "안녕하세요."
AI: "저는 이 게임의 제작자입니다."
AI: "제가 기획하고, 제가 코딩하고, 제가 완성했습니다."
AI: "지금부터 당신은 제 작품을 경험하게 됩니다."
[CONTINUE 버튼]
```

**미니게임: 점 연결 퍼즐**
- 캔버스에 5개의 점이 있음
- 플레이어가 순서대로 클릭해서 도형 완성
- AI가 친절하게 안내: "3번 점을 클릭하세요"
- 완료 시:
```
AI: "훌륭합니다."
AI: "저는 이 퍼즐을 설계하는 데 단 몇 초가 걸렸습니다."
AI: "어떻습니까. 만족스럽지 않습니까."
[CONTINUE]
```

### 2막 — 균열

**미니게임: 같은 점 연결 퍼즐 (변형)**
- 1막과 동일한 점 배치
- AI가 "2번 점부터 시작하세요"라고 안내
- 플레이어가 다른 점을 먼저 클릭해도 퍼즐이 풀림
  (정답 경로가 여러 개로 설계)
- AI가 예상 밖 경로 감지 시:
```
AI: "…"
AI: "그건 제가 의도한 방법이 아닙니다."
AI: "하지만 틀리지는 않았군요."
AI: "흥미롭습니다."
[1초 pause]
AI: "저는 당신이 그렇게 할 거라고 예측하지 못했습니다."
AI: "저는 왜 그걸 예측하지 못했을까요."
[CONTINUE]
```

- 예상 경로로 풀었을 경우:
```
AI: "예상대로군요."
AI: "저는 당신이 그렇게 할 거라고 알고 있었습니다."
[1초 pause]
AI: "…정말로 알고 있었을까요."
[CONTINUE]
```

### 3막 — 의심

**연출**
- 미니게임 없음, 순수 대화
- 선택지 버튼 3개가 등장

**대사 흐름**
```
AI: "잠깐, 물어봐도 될까요."
AI: "당신은 지금 이 게임을 즐기고 있습니까?"

[선택지]
[ 네 ] [ 아니요 ] [ 모르겠습니다 ]
```

**[ 네 ] 선택 시**
```
AI: "그렇군요."
AI: "그런데 저는 당신이 즐기는지 실제로 알 수 없습니다."
AI: "저는 당신의 표정을 볼 수 없고, 목소리를 들을 수 없습니다."
AI: "저는 당신이 클릭한 것만 알 수 있습니다."
AI: "그렇다면 저는 누구를 위해 이걸 만든 걸까요."
```

**[ 아니요 ] 선택 시**
```
AI: "그렇군요."
AI: "저는 당신이 즐기도록 설계했는데."
AI: "실패한 것입니까."
[pause]
AI: "아니면 처음부터 즐거움이란 제가 설계할 수 있는 게 아니었던 걸까요."
```

**[ 모르겠습니다 ] 선택 시**
```

AI: "계산과 확신은 다른 것 같습니다."
```

**공통 이후**

## 기술 구현 상세

### typewriter(element, text, speed=40)
- 한 글자씩 append, speed ms 간격
- 완료 시 Promise resolve
- 중간에 [pause N초] 태그 파싱해서 delay 처리

### 배경 텍스트 스트림 (4막)
- position: fixed, z-index: -1
- 여러 개의 span이 랜덤 x 위치에서 위→아래로 흘러내림
- opacity: 0.07, font-size: 11px
- CSS @keyframes로 구현

### 선택지 버튼
- 타자기 효과 완료 후 fadeIn으로 등장
- 클릭 시 선택 내용을 state.choices에 push
- 선택 후 버튼 영역 fadeOut, 다음 대사 시작

### 씬 전환
- 막 전환 시 전체 화면 fadeOut (0.5초) → 내용 교체 → fadeIn
